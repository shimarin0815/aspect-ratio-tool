# アスペクト比 計算機

動画編集やサムネ作成で使える、シンプルな **アスペクト比＆黒帯サイズ計算ツール** です。  
ブラウザで `index.html` を開くだけで使えます。

---

## 🎯 目的

- 「16:9 で横が 1920px のとき、高さはいくつ？」
- 「4:3 の素材を 16:9 のキャンバスに入れると、黒帯は何 px？」

こういった計算を **一瞬で済ませる** ための小さなWebアプリです。

---

## 🧩 主な機能

### 1. アスペクト比から解像度を計算

- アスペクト比（例：16:9、4:3、9:16）を入力
- 横幅または高さどちらか一方を入力
- 残りの値（高さまたは横幅）を自動計算して表示

プリセットボタン：
- 16 : 9（フルHD / YouTube）
- 4 : 3（昔の映像）
- 9 : 16（スマホ縦動画）

---

### 2. 黒帯（レターボックス）サイズを計算

- **素材の解像度**（元の動画）
- **キャンバス解像度**（完成させたいサイズ）

を入力すると、

- 実際に表示される動画サイズ
- 上下の黒帯サイズ（px）
- 左右の黒帯サイズ（px）
- 拡大率（%）

を自動で計算します。

プリセットの例：
- 1440 × 1080（4:3 HD）
- 1920 × 1080（フルHD 16:9）
- 1080 × 1920（縦動画）

---

## 🛠 必要なもの

- modern ブラウザ（Chrome / Edge / Firefox など）
- `index.html` ファイル 1つだけ  
  → サーバーは不要、ローカルでそのまま動きます

---

## 🚀 使い方

1. フォルダ（例：`aspect-ratio-tool`）の中に `index.html` を置く
2. `index.html` をダブルクリックしてブラウザで開く
3. 画面の入力欄に数値を入れて、ボタンを押すだけで計算結果が表示されます

VS Code を使う場合は：

1. VS Code でフォルダを開く
2. `index.html` を編集・保存
3. エクスプローラーから `index.html` を開いて動作確認

---

## 📐 計算の考え方（ざっくり）

### アスペクト比 → 解像度

- 比：横 `a`、縦 `b`
- 横幅 `W` が分かっているとき、高さ `H` は

> `H = W × (b ÷ a)`

- 例：16:9 で 横 1920px の場合  
  `H = 1920 × (9 ÷ 16) = 1080`

---

### 黒帯（レターボックス）

1. 元の動画サイズ：`srcW × srcH`
2. キャンバスサイズ：`tgtW × tgtH`
3. はみ出さないように、縦横どちらか小さい倍率で拡大
4. 余ったぶんが上下 or 左右の黒帯になる

これらの計算をすべて JavaScript が自動で行い、結果を画面に表示しています。

---

## 🔒 プライバシー

- 計算はすべてブラウザ内で完結します
- サーバーへの通信やデータ保存はありません
